## Prerequisites

* Docker 19.03+
* Docker Compose 1.25+

## Build Docker Image

```sh
./mvnw -Pprod jib:dockerBuild clean verify
```

## Installation

```sh
docker-compose up -d 
```

Then navigate to http://localhost:9090 in your browser.
