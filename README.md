## Prerequisites

* Docker 19.03+
* Docker Compose 1.25+

## Build Docker Image

```sh
./mvnw -Pprod jib:dockerBuild celan verify
```

## Installation

```sh
docker-compose up -d 
```

## Used Technologies

* Jhipster
* Spring Boot
* React
* Postgresql
* Redis
* Elasticsearch
